Praj Industries · 2023–Present ⭐ Flagship Project

AutoCAD Automation Plugin
Evaporation Heat Exchanger Drafting Engine

Problem

Mechanical engineers spent hours manually drafting complex heat exchanger components — General Arrangement Drawings, Tube Sheet Layouts, and Weld Fabrication Drawings — with high risk of human error in precision manufacturing specifications.

Solution

Designed and engineered a C# .NET plugin for AutoCAD 24 that takes parametric inputs (Shell ID, knuckle radius, tube pitching, baffle specs) and algorithmically generates all three drawing types with 100% manufacturing accuracy — replacing hours of manual drafting with seconds of computation.

Impact

Eliminated manual drafting errors entirely. Reduced engineering design cycle time by 80%+. Enabled junior engineers to produce production-ready drawings without senior oversight.

C# .NET AutoCAD 24 API Syncfusion SQL Server 2022 Azure DevOps
Praj Industries · 2023–Present Security / Automation

QR Code Document Verification
Automated Secure Stamping System

Problem

Industrial document workflows required tamper-proof verification stamps embedded precisely within PDFs — with no scalable way to handle concurrent high-traffic document uploads.

Solution

Built a hybrid C# + Python automation engine that locates specific text anchors within PDFs and programmatically embeds high-resolution QR codes at precise coordinates. Implemented thread-safe semaphore throttling to manage concurrent Python process execution safely.

Impact

Zero-touch secure document verification pipeline. Handles high-concurrency uploads without performance degradation. Eliminated manual stamp workflows entirely.

C# .NET Python iTextSharp QRCoder PyMuPDF SQL Server
